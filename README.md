# HowTo
## Setup
1. Setup using pip
```bash
python -m venv .venv && \
. .venv/bin/activate && \
pip install --upgrade pip && \
pip install -r requirements.txt
```
2. setup using poetry
```bash
python -m venv .venv && \
. .venv/bin/activate && \
poetry install
```
